## Installation

- ```git clone https://github.com/grandeto/consumer-manager.git```
- ```git checkout <desired branch>```

1. ```composer install```
2. Create DB
3. Setup .env
4. ```php artisan migrate:refresh --seed```
5. ```npm install```
6. ```npm run dev``` or ```npm run prod```
